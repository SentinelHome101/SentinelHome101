SentinelHome101 — Changelog


Version 1.0.1  ·  June 14, 2026
Feature and bug fix update.

Settings
- HIBP API key field added to Reports & Settings tab
- Users can now save their HaveIBeenPwned API key directly in the app
- Key is stored locally and never transmitted outside the machine
- Get a free key at: haveibeenpwned.com/API/Key

Bug Fixes
- Mouse wheel scrolling now works reliably on all tabs after a scan
  completes

Security
- Executable signed with a verified Microsoft Artifact Signing
  certificate
- Signature is permanently timestamped — valid beyond certificate
  expiry
- Windows UAC prompt now displays verified publisher name instead
  of Unknown Publisher
- VirusTotal scan result: 71 of 71 engines clean on signed build

Application
- Version number updated to 1.0.1 across all components

Website
- Standalone FAQ page added at sentinelhome101.com/faq.html
- Blog section added to main navigation and footer
- Version references updated site-wide

Known Limitations
- HaveIBeenPwned email check requires a free API key from
  haveibeenpwned.com
- Router firmware version detection depends on router exposing
  version in HTTP headers
